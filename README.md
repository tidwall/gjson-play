# GJSON Playground

A playground for [GJSON](https://github.com/tidwall/gjson).
Runs in the browser. 

[Live Demo](https://gjson.dev/)

## Build

```
make
```

## Test 

```
make serve
```

Open http://localhost:8000/ to test.
